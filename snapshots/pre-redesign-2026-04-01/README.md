# TTM Tutoring Website

Two-page premium private tuition marketing site built as a lightweight static front-end.

## Files

- `index.html`: conversion-focused home page
- `case-studies.html`: proof and credibility page
- `assets/styles.css`: shared design system and responsive styling
- `assets/site.js`: shared header/footer components and small UI enhancements

## Edit

- Update page copy directly in `index.html` and `case-studies.html`
- Update shared brand details, navigation, or footer contact in `assets/site.js`
- Adjust colors, spacing, typography, and layout rules in `assets/styles.css`

## Preview locally

Run a simple static server from the project root:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deploy

This is a static site. You can deploy the project root directly to:

- Netlify
- Vercel static hosting
- Cloudflare Pages
- GitHub Pages

No build step is required.

## Contact note

The site currently uses a visible placeholder email address for enquiries. Replace the contact address in `assets/site.js`, `index.html`, and `case-studies.html` when you are ready to launch.
